FizzBuzz

"FizzBuzz" a small 'infinite counter' that fizzes on numbers divisible by 5, buzzes on numbers divisible by 3, and fizzbuzzes when numbers are divisible by both. They give these at programming interviews and it piqued my curiosity. Go to PasteBin

Show/Hide Text in HTML

A Small Jquery script that shows/hides text. It is useful when attempting to hide text within a single div tag. I found it even more useful when my slider was reliant on switching images (using img tags), and thus pages. This would allow for text to show and hide underneath each img in the slider. However, since I've replaced the slider (on projects page) with a jquery slider I no longer needed the text. Another weakness is that the Jquery script is coded into the HTML itself, rather than at the bottom of the page. Go to PasteBin

"7Wars"

A small text adventure game made with python. To run this you need to install python, move the file into your chosen directory and run the command "python 7wars.py". Go to PasteBin

"Valley of Magog"

This is another small text game just download the files, install python, and run "python ValleyofMagog.py". Go to PasteBin
